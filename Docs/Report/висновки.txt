ВИСНОВКИ

У процесі виконання курсової роботи було успішно спроєктовано та реалізовано інформаційну систему бронювання репетиційних баз та студій звукозапису «SoundSpace». Система охоплює та автоматизує ключові бізнес-функції музичних майданчиків: ведення та фільтрацію каталогу послуг, онлайн-бронювання просторів, облік додаткового обладнання, а також строге розмежування прав доступу для клієнтів (музикантів) та адміністраторів (власників студій).

На першому етапі роботи було проведено аналіз предметної області, в якому виявлено основні проблеми сучасних студій звукозапису: конфлікти у розкладах через ручний облік, складнощі з орендою додаткового специфічного обладнання, а також відсутність зручної платформи для пошуку вільних музичних площ клієнтами. Впровадження централізованої системи з автоматизованими перевірками (зокрема щодо накладання часових інтервалів) дозволило усунути ці недоліки та суттєво покращити клієнтський досвід.

На основі результатів аналізу були сформульовані ключові вимоги до системи, що охоплюють як адміністративні, так і користувацькі аспекти: гнучкий пошук студій, управління власним розкладом, фіксування погодинної вартості апаратури, а також автоматична генерація фінансової статистики.

Під час логічного та концептуального проєктування було розроблено структуру реляційної бази даних, яка зберігає ключові сутності: студії, кімнати, обладнання, користувачів та бронювання. Реалізовано різноманітні зв’язки зберігання контенту та довідників. Для забезпечення консистентності даних на рівні бази були впроваджені тригери та збережені процедури у СУБД PostgreSQL, зокрема для перевірки перетинів бронювань та консистентного розрахунку фінальної вартості кожної сесії.

На етапі реалізації програмного коду було створено модульну архітектуру, яка відповідає сучасним індустріальним стандартам:
– Серверна частина (REST API) розроблена на базі платформи ASP.NET Core 8 з використанням мови C#. Взаємодія з базою даних забезпечується через Entity Framework Core, що дозволяє використовувати потужності об'єктно-реляційного відображення та міграцій. Авторизація та безпека реалізована за допомогою механізму JSON Web Tokens.
– Клієнтська частина, для якої функціонує серверне API, створена на основі фреймворку React 18 з використанням TypeScript та Tailwind CSS, що гарантує швидкий, надійний та кросплатформний користувацький інтерфейс для планування репетицій.

Окрім цього, для ефективного тестування системи було розроблено програмний генератор даних (Seeder), який програмно ініціалізує та заповнює БД сотнями тестових студій та тисячами імпровізованих бронювань, дозволяючи миттєво оцінити роботу системи під навантаженням.

Результатом курсової роботи є повноцінний програмний продукт «SoundSpace», який вирішує нагальні завдання управління музичними базами. Розроблена система демонструє високу стабільність, гнучкість масштабування і готова до подальшого розвитку або впровадження нових інтеграцій (наприклад, платіжних порталів чи програм лояльності).
